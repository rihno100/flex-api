justify-content

적용영역 : flex container
역할 : item 과 container 사이 남은 여백을 분할 또는 분산
기본값 : flex-start(남은 여백을 모두 뒤로보낸다.)
flex-end : 남은 여백을 모두 앞으로보내고 item 시작을 뒤에서부터 한다.
center :남은 여백을 좌우로 보내고 item 시작을 가운데서 한다.
space-between : container 와 item 사이여백을 제외하고 item간 여백을 균등하게 나눈다.
space-around : 남은 여백을 item을 감싸는 여백값으로 균등하게 나눈다.
space-evenly : 남은 여백을 item을 감싸는 여백값으로 균등하게 나눈다.(container 와 item사이 여백도 포함하여 균등분배) 

code
.container {
justify-content: flex-star, flex-end, center, space-between, space-around, space-evenly;
 }

algin-content

적용영역 : flex container
역할 : item 한 행별로 그룹화(content)하여 새로정렬
기본값 : stretch
flex-start : 상단에서 시작,남은 여백을 아래로 보낸다.
flex-end :하단에서 시작, 남은 여백을 위로보낸다.
center : 새로 중간 정렬, 남은 여백을 위아래로 보낸다.
stretch : 그룹별 container 높이만큼 균등하게 높이를 갖는다. 여백을 없앤다.
space-between : 남은 여백을 그룹여백값으로 균등하게 나눈다.(그룹과 container 사이여백은 제외)
space-around : 남은 여백을 각 그룹을 둘러싸게 균등하게 나눈다. 

code
.container {
align-content : stretch, flex-star, flex-end, center, space-between, space-around;
 }

flex-grow

적용영역 : flex item
역할 : 개별 item width 값을 정한다.
기본값 : 0
(flex-shrink의 기본값은 1이므로 flex-basis설정시 container가 줄어들때 basis 값 미만이 되면 shrink 비율로 줄어든다.)
단, flex-wrap : wrap 일경우 basis 값이 되면 item이 아래로 떨어진다. basis 설정 없이 flex-grow와 flex-shrink를 동실 설정시 우선 순위는 grow다.
만약, flex-basis 값이 있고 cotainer크기가 item들의 너비값보다 클경우 container너비에서 basis값을 제외한 나머지여백을 grow값으로 나누어 각 item의 너비값에 반영한다.
입력값 (정수) : ...-2,-1,0,1,2... 

code
.items {
flex-grow : ..-2,-1,0,1,2...;
 }

flex

적용영역 : flex item
역할 : item의 flex-grow, flex-shrink, flex-basis를 하나의 프로펄티로 변경
기본값 : flex: 0(flex-grow),1(flex-shrink),auto(flex-basis)
입력값 : flex-grow, flex-shrink, flex-basis 입력 값과 동일하게 입력 

code
.items {
flex : 0 1 auto;
 }
